#mrbs tablet

Ipad tablet optimised app to book a room

### requirments
docker, docker-compose, dinghy, bower, node, npm


##Configuration
Update the params in the docker-compose.yml file

##set up
```npm install && npm run setup```

##build assets
```npm run build```

##watch assets
```npm run build```

##run
```docker-compose up -d```
